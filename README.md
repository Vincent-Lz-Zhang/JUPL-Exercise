# JUPL Exercise

Instructions:

For unknown reason, you need to modify systemjs.config.js, to run unit test.

Modify:

paths: {
      'npm:': '/node_modules/'
    }
	
To be:
paths: {
      'npm:': '/base/node_modules/'
    }

And then you can run command:
npm test
